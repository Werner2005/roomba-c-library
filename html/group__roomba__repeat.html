Ausführliche Beschreibung

Definiert eine API um eine Befehl zyklisch immer wieder auszuführen.

Makro-Dokumentation

#define ROOMBA_REPEAT_ERROR_NOT_STARTED   -2

Thread konnte nicht gestartet werden

Dokumentation der Funktionen

int8_t roomba_repeat ( )

Funktion die immer wieder aufgerufen werden muss Diese Funktion sendet das mit roomba_repeat_start definiert Kommando Diese Funktion muss von dem Thread aufgerufen werden

Rückgabe
Funktion kann folgende Werte zurückgeben:
  • 0 kein Fehler
  • ROOMBA_REPEAT_ERROR_NOT_STARTED System ist nicht gestartet
