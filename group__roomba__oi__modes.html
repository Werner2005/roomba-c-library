Makro-Dokumentation

#define ROOMBA_SENSOR_OI_MODE_FULL   3

Full Mode

Im Full Mode akzeptiert der Roomba alle Befehle. Die interne API ist deaktiviert!

#define ROOMBA_SENSOR_OI_MODE_OFF   0

Off Mode

In diesem Modus nimmt der Roomba nur den ROOMBA_START Befehl an

#define ROOMBA_SENSOR_OI_MODE_PASSIVE   1

Passiv Mode

In diesem Modus nimmder Roomba alle Sensor Befehle an, den ROOMBA_SONG und alle Mode Befehl an

#define ROOMBA_SENSOR_OI_MODE_SAFE   2

Safe Mode

Im Safe Mode akzeptiert der Roomba alle Befehl

Die interne Sotware wechselt automatisch in den ROOMBA_SENSOR_OI_MODE_PASSIVE wenn ein Kritischer Zustand Auftritt
