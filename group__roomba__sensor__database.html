Ausführliche Beschreibung

Libary zu Aufbewahrung von Sensor Daten

Libary kann verwent werden um zu erkennen ob Daten vom Roomba neu

Daten werden eingelagert mit roomba_sensor_database_receive()

Dokumentation der Funktionen

int8_t roomba_sensor_database_get ( uint8_t  sensorID,
roombaSensorData_t *  result 
)

Liefert einen Bestimmte SensorID zurück

Wenn data->sensorID == ROOMBA_SENSOR_DISTANCE || data->sensorID == ROOMBA_SENSOR_ANGLE dann wird der Wert automatisch auf 0 gesetzt wenn es einmal abgerufen worden ist!

Rückgabe
Folgende Werte können zurück gegeben werden:
  • 0 Daten vorhanden
  • -1 Daten unbekannt

int8_t roomba_sensor_database_receive ( roombaSensorData_t *  data)

Daten im Internen Buffer einlagern Überschreibt bei Aktualisierung den Internen Buffer und meldet zurück ob die Daten neu waren!

Ob die Daten neu sind wird über den Vergleich 

 bufferData->data == data->data && bufferData->sensorID != 0

entschieden

Wenn data->sensorID == ROOMBA_SENSOR_DISTANCE || data->sensorID == ROOMBA_SENSOR_ANGLE wird immer angenommen das der Wert neu ist außer der wert war 0 und ist 0!

Rückgabe
Folgende Werte können zurück gegeben werden:
  • 0 Daten alt nicht aktualisiert
  • 1 Daten neu aktualisiert
  • -1 Übergebende Daten fehlerhaft
